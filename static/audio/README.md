- peaceful_village.mp3 
- mysterious_forest.mp3
- dark_ruins.mp3
- epic_mountains.mp3
- tavern_music.mp3
- royal_castle.mp3
- cave_ambience.mp3
- ocean_waves.mp3

Players should add their own royalty-free music files with these filenames.
